Installation
------------

In order to install the flatsurf package in Sage you need to run the following command::

    $ sage -p http://www.labri.fr/perso/vdelecro/flatsurf-VERSION.spkg

Where VERSION should be appropriately set to the version you want. You can also obtain
the current version of the code source with::

    $ git clone https://daemon@git.math.cnrs.fr/anon/plm/delecroix/flatsurf
