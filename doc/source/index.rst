Installation
------------

In order to install the surface_dynamic package in SageMath you need to run the
following command in a console::

    $ sage -sh -c "pip install surface_dynamics --user"

You can also obtain the current version of the code source with::

    $ git clone https://daemon@git.math.cnrs.fr/anon/plm/delecroix/flatsurf

or from github at https://github.com/videlec/flatsurf-package
